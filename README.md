# DevGrid - Test iOS sample

The DevGrid - Test iOS sample app is a Gist Comment application.

---
# Requirements

- iOS 14.1+
- Xcode 12.4+
- Swift 5.0

---

# Installation

The DevGrid - TestiOS SDK is added as a dependency using [CocoaPods](http://cocoapods.org).
Setup the workspace using:

```bash
$ pod install
```
